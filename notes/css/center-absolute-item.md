## How to center a absolute item

### x axis and y axis
```
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, 0);
```
### x axis
```
    position: absolute;
    left: 50%;
    transform: translate(-50%, 0);
```
### y axis
```
    position: absolute;
    top: 50%;
    transform: translate(-50%, 0);
```
