#### What Is Padding
Padding is the space between the [[border]] and the HTML element.

#### CSS Elements
padding: all;  
padding: up/bottom left/right;  
padding: up right bottom left;  
padding-top: top;  
padding-right: right;  
padding-bottom: bottom;  
padding-left: left;
