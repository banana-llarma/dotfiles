### What Is A Class Component

A class component is when you use a class to make a component

### [Syntax](Syntax)

Class based component syntax

```javascript
Class <component name> extends React.Component {
  render() {
    return (
      <div>
        put your tags hear
      </div>
    )
  }
}
```
